PORT REF

WELL-KNOWN PORT REFERENCE

    BULK ANNOTATE — PASTE DOCKER-COMPOSE OR K8S YAML

    SUPPORTED FORMATS
    • docker-compose block: - "8080:80" or - 8080:80/tcp
    • docker-compose IP-bound: - "127.0.0.1:8080:80"
    • docker-compose single port: - 80 or - 80/tcp
    • docker-compose inline array: ports: [80, 443] or ports: ["80:80"]
    • k8s scalar: port: 80 or ports: 8080
    • k8s container: containerPort: 80
    • k8s host: hostPort: 30080
    • k8s target (numeric): targetPort: 8080

    Only ports in the database (80+ well-known ports) are annotated. Unknown ports pass through unchanged.
