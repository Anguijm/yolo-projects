ip-cidr — IP/CIDR Calculator & Subnet Analyzer

Dissect a subnet → Check if an IP is inside → Split into smaller subnets → Compare multiple CIDRs

All four tools are visible below. Edit the CIDR to see everything update live.

Dissector
enter a CIDR block to see its components
Format: ipv4/prefix, prefix 0–32 (e.g. 10.0.0.0/8, 192.168.0.0/16)
Hover any bit for tooltip. Cyan = network prefix, dim = host bits, | = boundary
In-Range Checker
is an IP inside this subnet?
Checks whether this IP falls inside the CIDR above
Subnet Splitter
divide this subnet into smaller ones
Must be a strictly larger prefix than the current CIDR (smaller subnets)
Bulk Annotator
paste a list of CIDRs, get overlap detection
One CIDR per line. Max 256 lines. Overlapping pairs highlighted in amber.
