# Palette Gen

Color palette generator based on color harmony theory. Pick a seed color or hit spacebar for random, see 5 harmonious colors generated instantly. 5 modes: Analogous, Complementary, Triadic, Split-Complementary, Monochromatic.

## Features

- 5 color harmony modes with HSL-based generation
- Seed color picker or spacebar/tap to randomize
- Large full-height swatches with auto-calculated black/white text (WCAG luminance)
- Tap any swatch to copy hex to clipboard
- Lock individual colors to preserve them across regenerations
- Locked colors become the seed for new harmonies
- Export all colors as CSS custom properties
- Aesthetically constrained random seed (avoids washed-out colors)
- Mobile-first, full-viewport layout

## How to Run

Open `index.html`. Press spacebar or tap Generate for new palettes. Pick a seed color with the color picker. Switch harmony modes. Tap any swatch to copy its hex. Lock colors you like. Export as CSS variables.
