# Type Racer

Race against an AI ghost typist. Type the displayed text as fast as you can while the ghost progresses at a fixed WPM. Beat the ghost to win.

## Features

- Head-to-head race: your progress bar vs ghost progress bar
- Ghost speed: 30/45/60/80 WPM selectable
- Live WPM, accuracy, and time tracking
- Character-by-character validation (wrong keys flash red)
- 8 curated typing passages
- Paste prevention
- Win/lose result with final stats
- New Race button for instant replay
- OLED black, mobile-first, all PWA metas

## How to Run

Open `index.html`. Start typing to begin the race. Beat the ghost's progress bar. Select harder ghost speeds for a challenge.
