# readme-forge

Browser-based README.md generator. Fill a form, get a polished README with badges, TOC, and proper markdown structure. Zero dependencies, works offline.

## Features

- Form fields: project name, tagline, description, features, install steps, usage, tech stack, license, author, repo URL
- Live rendered markdown preview with inline renderer (no deps)
- Three templates: minimal, standard, detailed
- Badge generator: build status, license, version, PRs welcome, maintained
- Auto-generated table of contents (only non-empty sections)
- Copy markdown to clipboard with visual feedback
- Download as .md file named after project slug
- Offline, zero dependencies, IL2-safe

## Usage

Open `index.html` in any browser. No server required.

## Tech

Single HTML file. Vanilla JS. YOLO design system (dark industrial aesthetic).
