# Picross

Nonogram logic puzzle game. Deduce which cells to fill based on row and column number clues. Complete the grid to reveal pixel art. 15 handcrafted puzzles from 5x5 to 15x15.

## Features

- 15 puzzles: 5 easy (5x5), 5 medium (10x10), 5 hard (15x15)
- Fill/Cross toggle for marking cells
- Drag to fill/cross multiple cells in one swipe
- 3-strike mistake system with visual error feedback
- Clues gray out when a line is satisfied
- Pixel art color reveal on completion
- Progressive unlock — complete a puzzle to unlock the next
- localStorage saves progress across sessions
- Mobile-first with touch-action:none on grid, pointer events

## How to Run

Open `index.html`. Tap Play, select a puzzle. Use Fill mode to mark cells you think are filled, Cross mode to mark empty cells. Get the pattern right to reveal the hidden pixel art. 3 wrong fills and you lose!
