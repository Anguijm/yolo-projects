# Sound Synth

Browser-based synthesizer with keyboard input, waveform visualization, and reverb.

## How to run

Open `index.html` in a browser. Click keys or use your keyboard (A-K for notes, Z/X for octave).

## What you'd change

- Add ADSR envelope (sustain level)
- Add filter (lowpass/highpass with cutoff)
- Add recording/export to WAV
- Add a sequencer/looper
