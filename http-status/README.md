# http-status

Interactive HTTP status code reference. Browse all HTTP status codes (1xx–5xx) with descriptions, common causes, fix suggestions, and related headers. Search, filter by category, click for detail, copy as markdown. Fully offline, zero dependencies.

## Features

- **Full database**: All standard HTTP status codes 1xx through 5xx
- **Per-code detail**: Number, name, description, common causes, fix suggestions, related headers
- **Search**: Filter by code number or name/description text
- **Category filter**: Toggle 1xx / 2xx / 3xx / 4xx / 5xx
- **Expanded detail**: Click any card to expand full causes/fixes/headers
- **Copy as markdown**: Individual code or entire filtered set as markdown table
- **Cheat sheet mode**: Compact grid view for quick scanning

## Usage

Open `index.html` in any browser. No server required.

## Design

Follows the YOLO design system: dark terminal aesthetic, monospace font, ghost buttons, zero external dependencies.
