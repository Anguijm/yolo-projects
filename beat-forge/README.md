# Beat Forge

16-step drum machine with procedurally synthesized sounds. No audio samples — every drum sound is generated mathematically via Web Audio API oscillators, noise buffers, and filters. Shareable beats via URL encoding.

## Features

- 6 tracks: Kick, Snare, Closed Hi-hat, Open Hi-hat, Clap, Bass
- All sounds procedurally synthesized (sine sweeps, filtered noise, square wave bass)
- Lookahead audio scheduling for rock-solid timing
- Visual playhead with per-step highlighting
- BPM control (60-200) and swing percentage
- Weighted random beat generator (musically sensible patterns)
- Pattern encoded in URL hash — share beats by copying the link
- Sound preview on cell toggle
- Keyboard-friendly, mobile-first, OLED dark theme

## How to Run

Open `index.html`. Click cells to toggle beats. Press Play to hear your pattern. Hit Random for instant beats. Copy the URL to share your creation.
