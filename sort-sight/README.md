# Sort Sight

Sorting algorithm visualizer. Watch Bubble Sort, Insertion Sort, Selection Sort, Merge Sort, and Quick Sort animate step-by-step with colored bars and audio feedback.

## Features

- 5 sorting algorithms: Bubble, Insertion, Selection, Merge, Quick
- Animated bars with color-coded states: comparing (yellow), swapping (red), sorted (green)
- Audio tones mapped to bar values during comparisons
- Adjustable array size (10-120 elements) and animation speed
- Real-time comparison and swap counters
- Sorted completion sweep animation
- Safe abort: click Stop to interrupt any sort mid-run
- Controls disabled during sort to prevent state corruption
- Mobile-first, dark theme

## How to Run

Open `index.html`. Select an algorithm, press Sort. Watch the bars rearrange with color-coded operations and audio. Adjust size and speed with sliders. Hit Shuffle for a new array.
