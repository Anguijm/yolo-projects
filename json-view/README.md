# JSON View

JSON formatter and interactive tree viewer. Paste JSON, see it formatted with syntax highlighting, explore it as a collapsible tree, click keys to copy their JSON path.

## Features

- Collapsible tree view using native HTML5 details/summary elements
- Syntax highlighting for keys, strings, numbers, booleans, null
- Format (pretty-print) and Minify buttons
- Click any key in tree view to copy its JSON path to clipboard
- Smart path generation: dot notation for simple keys, bracket notation for special characters
- Graceful error reporting on invalid JSON
- One-click copy to clipboard
- Sample data button for instant demo
- Tab switching between Tree and Formatted text views
- Object/array item counts shown inline
- Dark IDE theme, monospace typography, mobile-first

## How to Run

Open `index.html`. Paste JSON into the input, or click Sample for demo data. Switch between Tree and Formatted views. Click any key name to copy its path. Use Format/Minify/Copy buttons.
