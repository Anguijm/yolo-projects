# Sprite Studio

Pixel animation editor. Draw pixel art frames on a grid, manage a timeline, preview animations with onion skinning, and export as PNG spritesheet. Zero dependencies.

## Features

- Configurable grid size: 8x8, 16x16, 32x32
- Pen, eraser, and flood fill tools with color picker
- Frame timeline: add, duplicate, delete, select frames
- Onion skinning: faint overlay of previous frame for smooth animation
- Real-time animation preview with adjustable FPS (1-24)
- PNG spritesheet export (all frames stitched horizontally)
- Responsive canvas sizing, checkerboard transparency indicator
- Mobile-first with pointer events and touch-action handling

## How to Run

Open `index.html`. Draw on the pixel grid, add frames with the timeline controls, enable onion skinning for smooth animation, preview with Play, and export your spritesheet as PNG.
