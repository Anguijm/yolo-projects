# Chaos Pendulum

Double pendulum simulation demonstrating chaotic motion. Trail traces paint mesmerizing patterns. Dual mode shows the butterfly effect — two pendulums starting 0.001 radians apart rapidly diverge.

## Features

- Double pendulum with Lagrangian equations of motion
- Fading trail traces the tip of the second arm
- Dual mode: two pendulums with tiny initial difference → chaos divergence
- Adjustable: arm lengths, masses, trail length
- Neon glow on masses, pivot point visualization
- Reset with random initial angles
- OLED black, mobile-first, all PWA metas

## How to Run

Open `index.html`. Watch the chaotic motion. Click Dual to see two nearly-identical pendulums diverge. Adjust lengths and masses. Reset for new patterns.
