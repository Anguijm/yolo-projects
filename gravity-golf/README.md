# Gravity Golf

Physics puzzle game. Slingshot-aim a ball through fields of gravity wells and repulsors to land on the target. 10 handcrafted levels with par scores, predictive trajectory preview, and neon space aesthetic.

## Features

- Slingshot aiming: pull back to set direction and power
- Dotted trajectory preview shows predicted path while aiming
- Gravity wells (blue, attract) and repulsors (orange, repel)
- 10 levels with escalating complexity and par scores
- Stroke counter with par/bogey/under-par rating
- Ball trail with fading opacity
- Glowing neon wells, green target rings, starfield background
- Responsive layout: positions scale with screen size without losing progress
- Mobile-first with pointer events

## How to Run

Open `index.html`. Pull back from the ball to aim (like a slingshot). Release to launch. Navigate around gravity wells to land on the green target. Fewer strokes = better score. Tap to retry on a miss.
