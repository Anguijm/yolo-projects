# Haiku Gen

Generative haiku poetry engine. Creates grammatically coherent 5-7-5 syllable poems using word banks organized by part of speech and syllable count. Seasonal themes shift the vocabulary and color palette.

## Features

- Strict 5-7-5 syllable structure via syntax templates
- Word banks: 200+ words organized by POS (adjective, noun, verb, preposition, determiner) and syllable count
- 8+ line templates per syllable count for variety
- 4 seasonal themes (Spring, Summer, Autumn, Winter) with color shifts and word biases
- Automatic "a" → "an" grammar correction before vowels
- Fade transition between poems
- Tap anywhere or press spacebar to generate
- Copy to clipboard, save favorites to localStorage
- Zen minimalist serif typography
- Duplicate word prevention within lines

## How to Run

Open `index.html`. Tap the screen or press spacebar to generate a new haiku. The background shifts with the season. Copy or save poems you like.
