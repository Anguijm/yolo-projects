# Graph Calc

Interactive function grapher. Type mathematical expressions and see them plotted on a zoomable, pannable coordinate plane. Multiple simultaneous functions with auto-assigned colors.

## Features

- Type any f(x) expression: sin(x), x^2, 1/x, tan(x), sqrt(x), abs(x), etc.
- Custom math parser with implicit multiplication (2x → 2*x)
- Supports: sin, cos, tan, asin, acos, atan, sqrt, abs, log (base 10), ln, exp, floor, ceil, pi, e
- Multiple functions with unique colors
- Pan by dragging, zoom with scroll wheel
- Pinch-to-zoom on mobile
- Dynamic grid that adapts to zoom level
- Asymptote detection (no vertical lines through discontinuities)
- Real-time cursor coordinates display
- 8 preset functions for instant demo
- Error highlighting on invalid expressions
- Dark theme, mobile-first

## How to Run

Open `index.html`. Type expressions in the input fields. Pan by dragging the graph, zoom with scroll wheel or pinch. Add more functions with the + button. Try the preset dropdown for instant examples.
