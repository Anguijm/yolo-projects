# Shadow Lab

Multi-layer CSS box-shadow generator with live preview. Add multiple shadow layers, adjust per-layer properties, preview on a customizable box, and copy production-ready CSS with one click.

## Features

- Multiple shadow layers: add, delete, reorder
- Per-layer controls: X/Y offset, blur, spread, color, opacity, inset toggle
- Live preview on customizable box (color, background, border-radius)
- 5 presets: Soft (layered), Hard, Neumorphism Light, Neumorphism Dark, Glow
- One-click CSS copy to clipboard
- Real-time updates as you adjust sliders
- Clean rgba() color output with opacity control
- Dark theme, mobile-first with scrollable controls

## How to Run

Open `index.html`. Adjust the shadow layers using sliders. Add more layers with "+ Layer". Try the presets for instant professional shadows. Customize the preview box color and background. Copy the CSS when done.
