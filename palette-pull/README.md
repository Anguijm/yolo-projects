# Palette Pull

Extract dominant colors from any image using k-means clustering. Upload a photo, get a beautiful color palette with hex codes and CSS variables.

## Features

- K-means clustering color extraction (4/5/6/8 colors)
- Pixel sampling grid for performance (100x100 max)
- Luminance-sorted palette output
- Click any swatch to copy hex code
- Click CSS block to copy all variables
- Dynamic text contrast on swatches
- Drag-and-drop or click-to-upload
- Sample image for instant demo
- Re-extract button for different random seeds
- OLED black, mobile-first, all PWA metas

## How to Run

Open `index.html`. Upload a photo or click Sample. Adjust color count. Click swatches to copy hex values.
